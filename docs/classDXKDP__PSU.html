Detailed Description

Class containing low-level functions to control the PSUs.

Uses input_message and output_message structs to construct messages based on function parameters. Uses CppLinuxSerial to communicate with the PSU through RS232 Serial.

Note: Exceptions are thrown using the THROW_EXCEPT macro, which is in the CppLinuxSerial Repo.
