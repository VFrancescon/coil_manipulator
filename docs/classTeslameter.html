Detailed Description

Uses SCPI and RS232 to communicate with the Teslameter.

We use SCPI to talk to the Teslameter (Lakeshore F71). as SCPI commands can be chained in a line, we set each command as a string
And the member functions combine them through string concatenation given a = "hello" and b = " world", c = a + b = "hello world"
